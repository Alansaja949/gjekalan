## Cara Menjalankan Aplikasi
Buka Android Studio → Open → pilih folder project aplikasi

Tunggu Gradle Sync selesai

Pilih device:

Emulator: Device Manager → Start Emulator

HP: Aktifkan USB Debugging → sambungkan ke laptop

Klik tombol ▶ Run

Tunggu proses instalasi → aplikasi berjalan
